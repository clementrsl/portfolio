# Portfolio — Clément Roussel

Portfolio BUT2 Science des Données (parcours EMS) — IUT de Vannes.
Site statique (HTML/CSS/JS, sans dépendance ni build).

## Voir en local
Double-cliquer sur `index.html` (ouverture dans le navigateur). Une connexion internet est utile pour les polices Google Fonts.

## Mettre en ligne sur GitHub Pages
1. Créer un dépôt GitHub (ex. `portfolio`).
2. Y déposer le **contenu de ce dossier** (`index.html` à la racine).
3. Dans le dépôt : **Settings → Pages → Branch : `main` / dossier `/root`** → Save.
4. Le site est publié sur `https://<ton-pseudo>.github.io/portfolio/`.

Astuce : pour une URL `https://<ton-pseudo>.github.io/` (sans sous-dossier), nommer le dépôt `<ton-pseudo>.github.io`.

## Structure
- `index.html` — la page
- `assets/css/style.css` — design system
- `assets/js/main.js` — interactions (nav, filtres SAÉ, lightbox, animations)
- `assets/img/figures/` — figures issues des rendus R
- `assets/docs/CV_Clement_Roussel.pdf` — CV

> Les visuels de l'alternance sont des **reconstitutions anonymisées** (aucune donnée réelle de l'entreprise).
